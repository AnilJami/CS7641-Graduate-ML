DataSet 1 Download: https://www.kaggle.com/jboysen/us-perm-visas/data
A stripped down version that contains only the necessary features is included in the /data folder

To run code (written in python 2.7):
0) navigate to jtay-code
1) For NNs run in jython: NN0.py, NN1.py, NN2.py and NN3.py
2) For continuous peaks run: jython continuouspeaks.py
3) For traveling salesman run: jython tsp.py
4) For flip flop run: jython flipflop.py
